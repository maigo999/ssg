言語紹介

PostCssの特徴

PostCssはJavascriptで動くCssのフレームワークだよー！
他のプリプロセッサと違ってPostCss自体にmixinやネストなど機能がはないから、プラグインを追加して自分の使いたい機能を実装していくものになってるよ！
PostCssはWebpackやGruntやGulpなど様々なタスクランナーで使うことができる。

このページでは主にglupを使った方法を教えるね！

gulpとは

GulpはNode.jsをもとに作られたタスクランナーでいろいろな作業をタスク化して自動化してくれる仕組みだよ！

インストール

STEP01

まずは前提となるNode.jsをインストールするよ！

ダウンロード先リンク

インストールが出来たらちゃんとインストール出来ているか確認するためにターミナルでこのコマンドを打つよ！

 
            
    node -v

正常にインストールされていれば下のようにバージョン情報が出るよ

                
    v10.14.2
    // インストール時点での最新版のバージョン情報が出ます。

STEP02

まずはGlupをインストールしていくよ。

ターミナルに下のコマンドを打とう！ 

                    
    npm install gulp -g

インストールができたらちゃんとできてるか確認！ターミナルに下のコマンドを打つよ！ 

                        
    gulp --v

インストールできていればこう表示されるよ！

                    
    [00:00:00] CLI version 2.0.1
    [00:00:00] Local version 4.0.0

これにてgulpのインストール作業は終了だよ！

STEP03

次にPostCssをインストールするよ！
これを入れないとPostCssをGulpで使えないから注意！


    npm install gulp-postcss --save-dev

STEP04

次にgulpの設定とpostcssの設定をするよ！
この設定では試しにpostcss-simple-varsを使ってみるね、postcss-simple-varsの詳細については下で解説するよ！
まずはgulpfile.jsというファイルを作るよ！

gulpfile.js


    const postcss = require('gulp-postcss');
    const gulp = require('gulp');
           
        gulp.task('css', function () {
        return gulp.src('main.css') //直下のmain.cssを探す。
            .pipe(postcss([ require('postcss-simple-vars') ]))//プラグインをタスクに追加
            .pipe(gulp.dest('./dest')); //destフォルダに出力
        });

次はpostcss.config.jsというファイルを作るよ！

postcss.config.js


    module.exports = {
        plugins: [
            require('postcss-simple-vars')
            //ここに入れるプラグインを記述
        ]
    }

これで設定は完了！

プラグイン紹介

変数

変数を使うためには"postcss-simple-vars"を入れよう。

インストールは下のコマンドを打とう。 

          
    npm install postcss-simple-vars


このプラグインを入れると$を使ったシンプルな変数が使えるよ。

                
    $main_color: #FCF6E5;
    $main_width: 960px;
                
    main{
        background: $main_color;
        width: $main_width;
    }

コンパイル結果はこうなるよ！

    
    main{
        background:#FCF6E5;
        width: 960px;
    }

条件文

条件文は一定の条件下で動く動かないを決める文なんだよ！
条件文を使うためには"postcss-conditionals"を入れよう。

インストールは下のコマンドを打とう。 

            
    npm install postcss-conditionals
            
          
            
    $check: true;
    .foo {
      @if $check == true {
        width: 100%;
      } @else {
        width: 50%;
      }
    }

コンパイル結果はこうなるよ！

            
    .foo {
        width: 100%
    }

ループ文

ループ文を使うためには"postcss-for"を入れよう。

同じものを何個も書かなくていいので便利！

インストールは下のコマンドを打とう。 

            
    npm install postcss-for

このプラグインを使うと@forを使ったループ文が書けるよ。

            
    @for $num from 1 to 3 {
        .fizz$num {
          width: $(num)px;
        }
    }

コンパイル結果はこうなるよ！

            
    .fizz1 {
        margin: 1px
    }
    .fizz2 {
        margin: 2px
    }
    .fizz3 {
        margin: 3px
    }
