言語紹介

Sassの特徴



ScssとはCSSプリプロセッサの一種です。
「Sassy CSS」を略したもので直訳すると「かっこいいCSS」「イカしたCSS」となります。
その名の通りCssを拡張したもので記述方法は基本的には同じです。
ですがネストという入れ子機能をつかって書けるので効率的に書くことができます。
実際に書いて見ます。

インストール

STEP01

まずは前提となるRubyをインストールしましょう。

ダウンロード先リンク

インストールが出来たら、インストール出来ているか確認するためにターミナルでこのコマンドを打ちます。

 
            
    ruby -v

正常にインストールされていれば下のようにバージョン情報が出ます。

                
    ruby 2.3.7p456
    // インストール時点での最新版のバージョン情報が出ます。

STEP02

それでは実際にSassをインストールしていきましょう。
ターミナルに下のコマンドを打ちましょう。 

                    
    gem install sass

インストールができたらちゃんとできてるか確認します。ターミナルに下のコマンドを打ちましょう。 

                        
    sass -v

インストールできていればこう表示されるはずです。

                    
    Ruby Sass 3.5.7

これにてSassのインストール作業は終了です。

機能紹介

変数

ここでは変数の書き方を教えるよ。
変数に値を入れておけば、変数名で値を参照することができるよ！ 

                
    $main_color: #FCF6E5;
    $main_width: 960px;

    main{
      background: $main_color;
      width: $main_width;
    }

コンパイル結果はこうなるよ！

    
    main {
      background: #FCF6E5;
      width: 960px;
    }

if文

if文は一定の条件下で動く動かないを決める文なんだよ！
Stylusではこういう風に書くよ！ 

            
    $check: true;

    .foo{
        @if $check == true {
            width: 100%;
        }@else{
            width: 50%;
        }
    }

コンパイル結果はこうなるよ！

            
    .foo {
        width: 100%
    }

for文

for文を使うと繰り返し処理を書くことができるよ！

同じものを何個も書かなくていいので便利！

            
    @for $i from 1 to 4 {
        .fizz#{$i} {
            width: $i + px;
        }
    }

コンパイル結果はこうなるよ！

            
    .fizz1 {
        margin: 1px
    }
    .fizz2 {
        margin: 2px
    }
    .fizz3 {
        margin: 3px
    }
