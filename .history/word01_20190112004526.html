言語紹介

Stylusの特徴

StylusはCSSプリプロセッサの一種だよ。
開発者はnode.jsやpugを作ったTJ Holowaychukさん！

他のプリプロセッサと違ってルールにあまりとらわれないプリプロセッサなんだ。
拡張子は".styl"を使うよ。
波括弧、セミコロン、コロン、カンマをつけてもつけなくてもコンパイルが通るから書き方を選ばないよ！
実際に書いてみるね！

Css

          
    .hoge {
        width: 100%
    }
          
    .hoge p {
        font-size: 24px
    }

Stylus

 
    .hoge
        width 100%
    p
        font-size 24px

こんな風にスッキリした記述ができるから可読性は高くなるし効率も上がるよ！

インストール

STEP01

まずは前提となるNode.jsをインストールするよ！

ダウンロード先リンク

インストールが出来たらちゃんとインストール出来ているか確認するためにターミナルでこのコマンドを打つよ！

 
            
    node -v

正常にインストールされていれば下のようにバージョン情報が出るよ

                
    v10.14.2
    // インストール時点での最新版のバージョン情報が出ます。

STEP02

それでは実際にStylusをインストールしていくよ！
ターミナルに下のコマンドを打とう！ 

                    
    npm install stylus -g

インストールができたらちゃんとできてるか確認！ターミナルに下のコマンドを打つよ！ 

                        
    stylus -V
    // Vは大文字じゃないと動きません。

インストールできていればこう表示されるよ！

                    
    0.54.5

これにてStylusのインストール作業は終了だよ！

機能紹介

Mixin

ここではMixinの書き方を教えるよ。
Mixinは別のところで定義したスタイルを別のところで呼び出せる機能だよ！
()をつけることでMixinを定義できるよ！ 

                
    fizz()
        font-size 20px
        border-radius 20px
              
    .buzz
        fizz()

コンパイル結果はこうなるよ！

    
    .buzz {
        font-size: 20px;
        border-radius: 20px;
    }

()の中に引数を入れることで値を渡すこともできるよ！


  
    foo(num)
        font-size num
        border-radius num
    .bar
        fizz(20px)

コンパイル結果はこうなるよ！

    
    .bar {
        font-size: 20px;
        border-radius: 20px;
    }

変数

ここでは変数の書き方を教えるよ。
変数に値を入れておけば、変数名で値を参照することができるよ！ 

                
    main_color = #FCF6E5
    main_width = 960px

    main
        background main_color
        width main_width

コンパイル結果はこうなるよ！

    
    main{
        background:#FCF6E5;
        width: 960px;
    }

条件文

条件文は一定の条件下で動く動かないを決める文なんだよ！
Stylusではifを使って、こういう風に書くよ！ 

            
    check = true

    .foo
        if check
            width: 100%
        else
            width: 50%

コンパイル結果はこうなるよ！

            
    .foo {
        width: 100%
    }

繰り返し文

繰り返し文は使うと繰り返し処理を書くことができる文だよ

同じものを何個も書かなくていいので便利！

Stylusではforを使って、こういう風に書くよ！

            
    for num in 1..3
        .fizz{num}
            margin num px

コンパイル結果はこうなるよ！

            
    .fizz1 {
        margin: 1px
    }
    .fizz2 {
        margin: 2px
    }
    .fizz3 {
        margin: 3px
    }
