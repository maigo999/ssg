インストール

STEP01

まずは前提となるNode.jsをインストールするよ！

ダウンロード先リンク

インストールが出来たらちゃんとインストール出来ているか確認するためにターミナルでこのコマンドを打つよ！

 
yarn
            
    node -v

正常にインストールされていれば下のようにバージョン情報が出るよ

                npm i -g|-D postcss-cli
    v10.14.2
    // インストール時点での最新版のバージョン情報が出ます。

STEP02

それでは実際にStylusをインストールしていくよ！
ターミナルに下のコマンドを打とう！ 

                    
    npm install stylus -g

インストールができたらちゃんとできてるか確認！ターミナルに下のコマンドを打つよ！ 

                        
    stylus -V
    // Vは大文字じゃないと動きません。

インストールできていればこう表示されるよ！

                    

                        /|\
                        //   //
                      //       //
                    //___*___*___//
                  //--*---------*--//
                /|| *             * ||/
              // ||*    v6.0.1     *|| //
            //   || *             * ||   //
          //_____||___*_________*___||_____//

これにてStylusのインストール作業は終了だよ！

機能紹介

変数

ここでは変数の書き方を教えるよ。
変数に値を入れておけば、変数名で値を参照することができるよ！ 

                
    main_color = #FCF6E5
    main_width = 960px

    main
        background main_color
        width main_width

コンパイル結果はこうなるよ！

    
    main{
        background:#FCF6E5;
        width: 960px;
    }

if文

if文は一定の条件下で動く動かないを決める文なんだよ！
Stylusではこういう風に書くよ！ 

            
    check = true

    .foo
        if check
            width: 100%
        else
            width: 50%

コンパイル結果はこうなるよ！

            
    .foo {
        width: 100%
    }

for文

for文を使うと繰り返し処理を書くことができるよ！

同じものを何個も書かなくていいので便利！

            
    for num in 1..3
        .fizz{num}
            margin num px

コンパイル結果はこうなるよ！

            
    .fizz1 {
        margin: 1px
    }
    .fizz2 {
        margin: 2px
    }
    .fizz3 {
        margin: 3px
    }
