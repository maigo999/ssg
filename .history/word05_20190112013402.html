言語紹介

Sass

Sassの特徴

SassとはCSSプリプロセッサの一種だよー。
「Syntactically Awesome Stylesheets」を略したもので直訳すると「かっこいいCSS」「イカしたCSS」になるよー。
記述が簡略化されるからコードがコンパクトになるよー。
それにネストっていう入れ子機能をつかって書けるから効率的に書くことができるよー。
じゃあ実際に書いて見るね。

Css


    .hoge {
        width: 100%
    }
          
    .hoge p {
        font-size: 26px
    }

Scss

          
    .hoge
        width: 100%;
        p
            font-size: 26px;

こんな感じにとってもコンパクトに書けるから効率もあがるよー

インストール

STEP01

まずは前提となるRubyをインストールするよー。

ダウンロード先リンク

インストールが出来たら、インストール出来ているか確認するためにターミナルでこのコマンドを打つよー。

 
            
    ruby -v

正常にインストールされていれば下のようにバージョン情報が出るよー。

                
    ruby 2.3.7p456
    // インストール時点での最新版のバージョン情報が出ます。

STEP02

それじゃあ実際にSassをインストールしていくよー。
ターミナルに下のコマンドを打とう！ 

                    
    gem install sass

インストールができたらちゃんとできてるか確認します。ターミナルに下のコマンドを打ちましょう。 

                        
    sass -v

インストールできていればこう表示されるはずです。

                    
    Ruby Sass 3.5.7

これにてSassのインストール作業は終了です。

機能紹介

Mixin

ここではMixinの書き方を教えます。
Mixinは別のところで定義したスタイルを別のところで呼び出せる機能です。
@mixinをつけることでMixinを定義でき、@includeで呼び出すことができます。 

                  
    @mixin fizz{
      font-size :20px;
      border-radius :20px;
    }

    .buzz{
      @include fizz;
    }

コンパイル結果はこうなります。

      
      .buzz {
          font-size: 20px;
          border-radius: 20px;
      }

()の中に引数を入れることで値を渡すこともできます。

    
    @mixin foo($num){
        font-size:$num;
        border-radius:$num;
    }
    .bar{
        @include foo(20px);
    }

コンパイル結果はこうなります。

      
      .bar {
          font-size: 20px;
          border-radius: 20px;
      }

変数

ここでは変数の書き方を教えます。
変数に値を入れておけば、変数名で値を参照することができます。 

                
    $main_color: #FCF6E5;
    $main_width: 960px;

    main{
      background: $main_color;
      width: $main_width;
    }

コンパイル結果はこうなります。

    
    main {
      background: #FCF6E5;
      width: 960px;
    }

条件文

条件文は一定の条件下で動く動かないを決める文です。
Scssではifを使って、このように書きます。 

            
    $check: true;

    .foo{
        @if $check == true {
            width: 100%;
        }@else{
            width: 50%;
        }
    }

コンパイル結果はこうなります。

            
    .foo {
        width: 100%
    }

繰り返し文

繰り返し文は使うと繰り返し処理を書くことができる文です。

同じものを何個も書かなくていいので時間の短縮になります。

Scssではforを使って、書くことができます。

            
    @for $i from 1 to 4 {
        .fizz#{$i} {
            width: $i + px;
        }
    }

コンパイル結果はこうなります。

            
    .fizz1 {
        margin: 1px
    }
    .fizz2 {
        margin: 2px
    }
    .fizz3 {
        margin: 3px
    }
